# 物联网设备管理页面

## 功能概述

这是一个简化的物联网设备管理页面，提供设备列表展示和基本管理功能。

## 主要功能

### 1. 设备列表管理
- **批量设备查询**: 支持分页查询设备列表
- **设备状态显示**: 显示设备连接状态（在线/离线/未激活/禁用）
- **设备详情展示**: 显示每个设备的详细信息
- **下拉刷新**: 支持下拉刷新设备列表
- **上拉加载**: 支持上拉加载更多设备

### 2. 设备统计信息
- **设备数量统计**: 显示总设备数、IoT设备数、在线设备数等
- **状态统计**: 实时统计各种状态的设备数量

## API接口集成

### 已集成的接口

1. **获取设备列表** (`GET /api/devices/`)
   - 支持分页查询设备列表
   - 可按状态筛选
   - 参数：page、page_size、type、include_deleted

## 使用方法

### 1. 页面跳转
```javascript
wx.navigateTo({
  url: '/pages/iotDevice/iotDevice'
});
```

### 2. 功能调用
页面加载后会自动：
1. 获取设备列表
2. 统计设备信息
3. 显示设备状态

### 3. 交互操作
- **下拉刷新**: 在设备列表页面下拉可刷新数据
- **上拉加载**: 在设备列表页面底部上拉可加载更多
- **设备点击**: 点击设备项可查看详细信息
